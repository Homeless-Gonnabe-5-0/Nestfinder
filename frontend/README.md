# NestFinder

A modern landing page for NestFinder - helping you find the best apartments in Ottawa.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React 19**

## Design

- Brand color: Forest Green (#1F4D2B)
- Clean, modern design inspired by yellowcake.dev
- Fully responsive layout
